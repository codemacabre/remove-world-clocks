# Remove World Clocks GNOME Extension

This is a minimal GNOME extension which removes the 'World Clocks' section from the date menu.

Install this extension directly from https://extensions.gnome.org/extension/6973/remove-world-clocks/, or to manually install, copy the `remove-world-clocks@codemacabre.com` folder to `~/.local/share/gnome-shell/extensions` and restart the GNOME Shell.
